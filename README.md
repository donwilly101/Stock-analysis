# Stock Analysis — Week 1 Project

This repository contains my **Week 1 submission** for the Data Science Stock Analysis project.  
It focuses on loading, cleaning, and exploring stock market datasets using Python.

---

##  Project Overview
This project explores stock market trends by analyzing multiple datasets:
- Company information  
- Daily stock prices  
- Market indices  
- Stock prices with technical indicators  

Each dataset helps build an understanding of how different companies and markets behave over time.
